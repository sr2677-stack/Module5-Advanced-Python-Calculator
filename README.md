# Advanced Calculator

## Features
- REPL interface
- Strategy Pattern
- Factory Pattern
- Observer Pattern
- Memento Pattern
- Facade Pattern
- Pandas history persistence
- Undo/Redo
- Config via .env
- Full test automation
- 100% coverage enforced

## Setup
pip install -r requirements.txt

## Run
python -m app.calculator_repl

## Run Tests
pytest --cov=app --cov-report=term-missing
